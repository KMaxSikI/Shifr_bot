Данный код создает Telegram-бота, способного шифровать и дешифровать текстовые сообщения, для обеспечения конфиденциальной переписки между пользователями.
За основу бота взята библиотека aiogram для работы с Telegram API.
Созданы следующие модули для упрощения читаемости кода:
1.	SQL_DB - Этот модуль, отвечает за взаимодействие с базой данных. Он реализует функции для сохранения, извлечения и управления данными, связанными с шифрованием и дешифрованием.
2.	Buttons - Этот модуль создает различные кнопки для пользовательского интерфейса в Telegram.
3.	Texts - Этот модуль, содержит текстовые сообщения, которые бот отправляет пользователям.
4.	Encrypt - Этот модуль отвечает за шифрование сообщений. Он, содержит функции или методы для преобразования входного текста в зашифрованный вид.
5.	Decrypt - Этот модуль, содержит функции или методы для дешифрования зашифрованных сообщений обратно в их исходное состояние.
